## tools.video とは？

**tools.video** は、**完全にブラウザ内で動作する** 18種類の無料オンライン動画・音声ツールです。ファイルはデバイスから一切外に出ません。最先端のWeb技術を活用し、すべてローカルで処理します。

**[tools.video](https://tools.video)** にアクセスするだけ。それだけです。

## 特徴

### 📦 最大50GBのファイルに対応

4K映像、8K RAWエクスポート、数時間の録画——どんな大きなファイルも処理可能。アップロード制限、サーバータイムアウト、画質劣化はありません。

### 🎞 あらゆるフォーマットに対応

**動画：** MP4、WebM、MKV、AVI、MOV、FLV、WMV、M4V、TS  
**音声：** MP3、WAV、FLAC、AAC、OGG、WMA、M4A、OPUS、AIFF、DTS、AC3、EAC3、APE、AMR、MKA  
5.1 / 7.1サラウンドを含むマルチチャンネルオーディオに完全対応。

### ⚡ 圧倒的な高速処理

デュアルエンジンアーキテクチャ：
- **MediaBunny** — WebCodecs APIによるハードウェアアクセラレーション
- **FFmpeg.wasm** — WebAssemblyベースの最大限のフォーマット互換性

マルチスレッド処理でデバイスのフルパワーを活用。

### 🔒 100%プライバシー保護

ファイルは**デバイスから一切外に出ません**。アップロードなし、サーバー処理なし、進捗バーの待ち時間なし。データは常にあなたのものです。

### 🚀 ゼロフリクション

ダウンロード不要、インストール不要、会員登録不要。ページを開いてファイルをドロップするだけ。

## プライバシー

私たちのプライバシーへの取り組み：

- **ゼロアップロード** — すべての処理はブラウザ内で完結
- **サーバー処理なし** — ファイルは常にデバイス上に保持
- **ファイル内容の追跡なし** — メディアの閲覧や分析は一切行いません
- **アカウント不要** — すべてのツールを匿名で利用可能

## 技術スタック

tools.video はモダンなWeb技術で構築されています：

- **[Next.js 15](https://nextjs.org/)** — SSR対応のReactフレームワーク（SEO最適化）
- **[MediaBunny](https://mediabunny.dev/)** — WebCodecsベースのメディア処理エンジン
- **[FFmpeg.wasm](https://ffmpegwasm.netlify.app/)** — FFmpegのWebAssemblyコンパイル版
- **[SharedArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/SharedArrayBuffer)** — ブラウザでのマルチスレッド処理
- **19言語対応** — グローバルユーザー向けのローカライゼーション
